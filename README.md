# Shader-Editor
Editor básico para manipular e criar items via shader

```bash
# Clone o repo
git clone https://github.com/gilfernandes234/Shader-Editor.git

# Instalar dependências
Rode o install.bat

# Executar o editor
execute o shader.py

ou no cmd:
py shader.py
```
